Team-Squash old_game.Assignment_5.Scrabble Game

How to run:
- open the terminal/command prompt
- run the following command
    java -jar Team_Squash.jar

alternatively, if your system supports it, you may be able to simply double click on the jar, as it is a GUI Application.

The program will guide you through everything else. For help, simply type "HELP" into the text box.

Dependencies:
- Java8 JDK with bundled JavaFX Package

This jar was tested on Windows 10 Home Edition and Linux Mint.
