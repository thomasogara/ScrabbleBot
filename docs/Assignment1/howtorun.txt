How to run old_game.Scrabble.jar

Make sure the /assets folder is present with a charset.conf file within it.
This charset.conf file contains the list of valid characters for a tile.
Note: the '0' tile represents a blank tile.

1. Navigate to the directory of old_game.Scrabble.jar

2. Execute the command ' java -jar old_game.Scrabble.jar '

3. Observe the output. And confirm all actions performed are correct;
